# 🛰️ חדר מצב: משבר הטילים בקובה (1962)

אפליקציית לימוד אינטראקטיבית לתלמידי כיתה ט׳ — שיעור היסטוריה חוויתי על משבר הטילים בקובה, בשילוב **מתודת FDE** (Forward Deployed Engineering) ותכנון סוכן AI.

האפליקציה בנויה ב־React + TypeScript + Vite, בעברית מלאה עם תמיכת RTL, ורספונסיבית למחשב ולמובייל.

## 🎯 מה יש באפליקציה?

מסע בן 10 תחנות בעיצוב "חדר מצב מודיעיני":

1. **מסך פתיחה** — היכרות עם המשבר והמשימה
2. **מבוא למתודת FDE** — הסבר פשוט על השיטה
3. **4 השלבים** — ציר זמן אינטראקטיבי של שלבי המודל
4. **תיק לקוח** — מברקים סודיים, תצלומי לוויין, קנדי, חרושצ׳וב ו־EXCOMM
5. **זיהוי צווארי בקבוק** — פעילות בחירה עם משוב מיידי
6. **תכנון סוכן AI** — טופס הגדרת שם, תפקיד, מקורות ידע, מגבלות ופלט
7. **כתיבת פרומפט** — עורך פרומפט מערכת עם צ׳קליסט חכם
8. **סימולציה** — מברק נכנס, ניתוח הסוכן ובחירת חלופת פעולה (נתוני דמה, ללא חיבור אמיתי ל־AI)
9. **רפלקציה וסיכום** — שאלות פתוחות לדיון כיתתי
10. **מה הרווחנו** — השוואה ויזואלית: למידה מסורתית מול FDE

## 🚀 הוראות הפעלה

דרישות מוקדמות: [Node.js](https://nodejs.org/) גרסה 18 ומעלה.

```bash
# התקנת תלויות
npm install

# הרצה בסביבת פיתוח (נפתח בכתובת http://localhost:5173)
npm run dev

# בניית גרסה לפרסום
npm run build

# תצוגה מקדימה של גרסת הפרסום
npm run preview
```

## 📝 הערות למורה

- כל הנתונים באפליקציה הם **נתוני דמה** המבוססים בקירוב על האירועים ההיסטוריים של אוקטובר 1962.
- אין חיבור אמיתי למודל AI — הסימולציה מדגימה כיצד סוכן היה נראה ופועל.
- ההגדרות שהתלמידים ממלאים במסך "תכנון סוכן" זורמות אוטומטית למסכי הפרומפט והסימולציה.
- תשובות הרפלקציה נשמרות על המסך בלבד (לא נשלחות לשום מקום) — מומלץ לשתף בדיון כיתתי.
